常用 Git 命令

以下是日常开发中最常用的几个 Git 操作：

  1. git clone — 克隆远程仓库到本地
  2. git add . — 暂存所有改动
  3. git commit -m "消息" — 提交并附说明
  4. git push — 推送到远程分支
  5. git pull — 拉取远程更新
